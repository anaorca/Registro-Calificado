# Curso: ESpecialización en la Construcción de Registro Calificado

Aprende a gestionar, construir y radicar exitosamente el **Documento Maestro** de programas académicos bajo los estándares del Decreto 1330 y Decreto 0529.

## Propuesta de Valor
Este curso proporciona la ruta técnica para que directivos y docentes cumplan con las **15 condiciones de calidad** y dominen el proceso administrativo ante el Ministerio de Educación Nacional.

## Mapa de Entrenamiento (5 Módulos)
1.  **Módulo 1: El Proceso Administrativo**: Cronograma legal y rutas de radicación (1330 vs 0529).
2.  **Módulo 2: 6 Condiciones Institucionales**: Preparando la base para la Pre-radicación.
3.  **Módulo 3: 9 Condiciones de Programa**: Guía técnica para redactar el Documento Maestro.
4.  **Módulo 4: Ensamblaje y SACES**: Cómo consolidar el expediente y navegar la plataforma oficial.
5.  **Módulo 5: Visita de Pares y Certificación**: Defensa del programa y evaluación final.
